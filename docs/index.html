Rain World Bestiary

The Bestiary is a mod that adds an encyclopedia of creatures to Rain World, this bestiary unlocks as you interact with creatures, only displaying information the user has hopefully figured out by themselves.

If you have a suggestion, want to report a bug or issue (including performance issues), or ask help with something, feel free to submit an issue request on GitHub, here

To find more on how to setup entries in your mod, you can check out the walkthrough. To find out how to make entries, you can check out JSON Components.
